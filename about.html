what is bookbear?

In continuing the programme and activity of
reading in the pandemic COVID-19 atmosphere, Sarawak State Library has
provided new services through BookBear Services.

BookBear is a direct book loan service to a user's home which users can download the Number app. sa2 via Google Play Store or App Store that will start operating on 13 June 2020 as below:

Operation Day: Monday to Saturday
Operation Hours: 9:00 am - 5:00 pm

This initiative is meant to ensure that users get access to the Library collection since Sarawak State Library has not yet fully opened the premises.
